JumpBack Icons CDN

This page hosts the SVG icons used by the JumpBack Figma plugin. You can link to any of the icons directly from your plugin UI or web projects.

Available Icons

  • Component Icon
  • Instance Icon
  • Group Icon
  • Frame Icon
  • Section Icon
  • Page Icon
  • Image Icon
  • Vector Icon
  • Text Icon
  • Default Icon

Usage Example

Include an icon in your HTML like this:

<img src="https://owlson.github.io/figmajumpback/assets/icons/component.svg"
     alt="Component Icon"
     width="20" height="20">
